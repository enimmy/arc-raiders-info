Gun Notes (assume level 4)

Renegade — Green Stock Silencer / Extended

  • Medium/long range beam. TTK is relatively slow but performs at relevant ranges.
  • Has recoil but not high firerate. Green stock helps bloom/recovery; blue stock reduces recoil recovery but is usually not worth it.
  • Compensator/muzzle break not needed. Prefer silencer or extended barrel.

Il Toro — Lightweight Stock Choke

  • Close range only; do not ADS.
  • Choke greatly reduces base bloom — increases effective range.
  • High recoil but low firerate so recoil doesn't matter; stocks other than lightweight are wasteful. Lightweight gives very strong handling buff.

Venator — Green Angled

  • Very fast TTK to close/medium. Don't ADS at close range.
  • Noticeable horizontal recoil. At medium ranges you'll ADS, so blue angled is not worth it.

Vulcano — Pink/Blue Angled Choke Kinetic Converter

  • Close range only; best bodyshot TTK. Do not ADS.
  • Choke reduces base bloom. High firerate and horizontal recoil — angled grip preferred (blue acceptable since not ADSing).
  • Lightweight stock is decent. Kinetic converter trades recoil for firerate (recommended).

Tempest — Compensator Green Angled

  • Solid TTK to close/medium. Fast firerate and high bloom growth — compensator extends effective range.
  • Both horizontal and vertical recoil present; angled grip preferred. Because it's used at medium ranges where you ADS, avoid blue grips.

Torrente — Kinetic Converter / Lightweight Silencer / Muzzle Break

  • Close range only; do not ADS. Excellent bodyshot TTK; headshots are RNG due to bloom.
  • Vertical recoil present; muzzle break helps slightly. Leave stock empty unless lightweight or kinetic is available.

Bobcat & Stitcher — Compensator Pink/Blue Angled Lightweight / Kinetic

  • Small base bloom that ADS reduces slightly; very high firerate and bloom growth so close range only.
  • Compensator is obvious; angled grip is preferred due to horizontal recoil. High-level grips acceptable because these guns are rarely ADSed.
  • Lightweight or kinetic are both strong choices.

Anvil — Compensator / Silencer / Extended

  • No grip slot. High per-shot bloom that doesn't reset fast; not suited for DPS brawls.
  • Compensator helps skilled players with faster follow-up shots; others should prefer silencer or extended barrel and avoid close-range DPS fights.

Ferro — Silencer / Extended Lightweight

  • High base dispersion; ADSing significantly improves accuracy. Recoil/bloom don't matter much because of reload cadence.
  • Use lightweight stock to improve mobility; grips/other stocks are mostly pointless.

General gun rules: guns are noted at level 4. It's usually better to bring a weaker gun that is level 4 than a "better" gun at level 1–3. Recycle/sell unnecessary attachments (shotgun silencer, vertical grips, padded stock, blue stock, anvil splitter). Don't put unnecessary attachments on guns — recycle them.

Tech & Tactics

  • Shotguns do no extra headshot damage — aim for body.
  • If a teammate pings an enemy you can't see, shoot at the ping (useful in dark/bushy areas).
  • If an Anvil/Ferro shot doesn't pop a shield, it was a medium/heavy shield.
  • Arc usually can't see through bushes/smoke unless already in red mode; sprinting affects it strongly.
  • Dodge-roll recovery can be canceled by crouching — allows faster shooting after a roll.
  • You can cancel equip/unequip animation with dodge-rolling. A fast switch technique: press to equip, instantly dodge, hold crouch, and tap-fire (spam M1) to begin shooting quickly.
  • Do not wear backpacks with light strips — they reveal you.
  • Bullets originate from the barrel position; you can sometimes shoot without exposing character model. Hip-fire benefits from barrel offset.
  • Dodging off a ledge then dodging at the bottom is faster than jumping then dodging from the bottom — you can shoot faster while avoiding some fall damage.

Monitor

  • Enable any "shadow boost" / "black equalizer" / "dark boost" feature — improves dark-visibility without destroying image.
  • If you need more visibility, adjust gamma (lower gamma can help), then raise contrast slightly. If monitor lacks gamma/contrast, adjust in the GPU control panel.

Graphics (NVIDIA)

Core settings summary
  • Window: Borderless
  • Frame Gen: Off
  • Vsync: Off
  • Reflex: On (boost preferred)
  • Limit: Unlimited or ~10 frames below your average
  • Scaling Method: DLSS
  • Quality: Balance visibility vs FPS (Transformer model recommended on 40/50 series)
  • RTX: Static for max frames; High otherwise
  • View Distance: High
  • AA: Low
  • Shadow: High (competitive advantage)
  • Post: Low, Texture: High/Epic, Effects: Low, Reflections: Epic, Foliage: Low, Illumination: Low
  • Energy savings: off (especially if G-Sync enabled)

Tip: use the newer transformer DLSS model on modern cards for clearer images; older cards use CNN or older transformer.

Audio

  • Night Mode: On — it normalizes audio levels so footsteps and quiet sounds become clearer relative to loud gunfire. Useful in quiet/more tactical play.
  • Night mode can be overstimulating for some players; try both and pick the one that gives clearer footsteps for you.

Publish: GitHub Pages

Instructions to host this single-file site on GitHub Pages. Two options: project site or user site.

Quick steps (project site)

  1. Create a repository on GitHub (example: my-guide).
  2. Place this file as index.html in the repository root (or in a /docs folder if you prefer).
  3. Commit & push the file to the default branch (commonly main).
  4. On GitHub: Repository → Settings → Pages. Under "Source" choose the branch (main) and root (or /docs) and Save.
  5. Your site is then published at https://<username>.github.io/<repo>/. For a user site use a repo named <username>.github.io and the site will be https://<username>.github.io/.

Command-line example

git init
git add index.html
git commit -m "Initial site"
git branch -M main
git remote add origin git@github.com:YOUR_USERNAME/REPO_NAME.git
git push -u origin main

Alternative: create the repo on github.com and upload index.html via the web UI (Add file → Upload files) then use Settings → Pages to enable publishing.

Notes and extras

  • Project site URL format: https://USERNAME.github.io/REPO_NAME/.
  • User site: repository must be named USERNAME.github.io. The root index.html serves at https://USERNAME.github.io/.
  • If you prefer the site in /docs folder, select the branch + /docs in Pages settings.
  • If you want a custom domain, add a CNAME file to repo root or set it via Pages settings; also update DNS to point to GitHub Pages (A / CNAME records).

Quick Reference

Top reminders
  • Blue grips: -30% ADS speed, +10% recoil. Avoid at medium ranges where ADS is used.
  • Shotguns: aim body — no headshot bonus.
  • Recycle: shotgun silencer, vertical grips, padded & blue stock, anvil splitter.
  • Weapon equip/unequip can be canceled with dodge roll + crouch + tap-fire technique.
